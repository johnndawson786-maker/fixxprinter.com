Detailed troubleshooting

Printer not working after trying the basics? Go deeper.

1

Restart everything in the right order

Power off the printer. Then restart the router. Wait 60 seconds. Power the printer back on. Wait for it to reach the idle state. Then restart the computer or device you are printing from. This clears most temporary network and spooler faults.

2

Clear the print queue

On Windows: open the Start menu, search for Services, find Print Spooler, stop it, delete all files in the spool PRINTERS folder, and restart the Print Spooler service. On Mac: go to Printers & Scanners, select the printer, and click Open Print Queue to cancel all jobs.

3

Remove and reinstall the printer

On Windows: open Settings, go to Printers & scanners, select the printer, and click Remove. Then add it again. On Mac: in Printers & Scanners, click the minus button to remove the printer, then click the plus button to add it again. This often resolves configuration corruption.

4

Reinstall the printer driver

Download the latest full driver package from the manufacturer official support website using your exact model number. Uninstall the existing driver first using the manufacturer uninstaller if one is provided. Reboot the computer and run the new installer. Choose the correct connection type during installation.

5

Contact support if the issue persists

If the printer still does not work after completing all steps, the issue may require remote diagnosis. Describe your printer model, operating system, connection type, and the exact error message when you contact us.

By symptom

Specific fixes for specific print quality problems.

Streaks on printed pages

Usually caused by a clogged printhead or dirty drum. Run the printhead cleaning utility from the printer maintenance menu 2 to 3 times, then print a test page.

For laser printers, streaks may indicate a worn drum or toner cartridge near empty.

Colors look wrong

Run a color calibration from the printer settings. Check that each color cartridge has ink. If using photo paper, confirm the paper type is set correctly in the print dialog.

Replace the cartridge of the color that looks missing or wrong.

Printing too slowly

Check if the printer is set to High Quality mode, which is slower. Switch to Normal or Draft for everyday documents. Also check if the USB or network connection is running at full speed.

Wireless printing is slower than USB. For large jobs, use USB or Ethernet.

Pages print crooked

Remove all paper from the tray and reload it, confirming the side guides are snug against the paper without squeezing. Also check that the paper is not curled or damp.

If the problem continues, clean the paper rollers with a lightly dampened cloth.
