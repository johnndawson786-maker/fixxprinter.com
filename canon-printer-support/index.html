Canon printer series

Which Canon printer series do you have?

📸

Canon PIXMA

The PIXMA line is Canon's home inkjet series covering everything from basic document printing to six-color photo output. Common issues include Wi-Fi disconnections, ink absorption errors on older models, and print head alignment after cartridge replacement.

⚡

Canon imageCLASS

Canon's laser printer line for home and small office use. imageCLASS models are fast, economical for text printing, and reliable in networked environments. Common issues include offline status after network changes and driver compatibility after macOS updates.

🏢

Canon MAXIFY

The MAXIFY line targets small businesses needing fast inkjet printing with high-yield cartridges. MAXIFY printers support automatic duplex and ADF scanning. Common issues include ink cartridge recognition errors and firmware-related Wi-Fi drops.

🛢️

Canon MegaTank (G series)

Canon MegaTank printers use large refillable ink reservoirs for very low per-page cost. The G series is ideal for frequent printing. Common issues include air in the ink system after refilling and printhead clogging after extended inactivity.

📷

Canon SELPHY

Compact photo printers using dye-sublimation for laboratory-quality photo output. SELPHY printers connect via Wi-Fi, USB, or direct memory card. Common issues include paper size mismatches and connection errors with the Canon PRINT app.

📟

Canon TR and TS series

Mid-range home inkjet AIO printers with Wi-Fi, auto duplex, and ADF options. TR and TS series models are among the most commonly owned Canon home printers. Common issues include offline errors on Windows 11 after Wi-Fi password changes.

Common Canon issues

Step-by-step fixes for the most reported Canon printer problems.

1

Canon printer shows offline

Open Printers and Scanners in Windows Settings or Mac System Settings. Select the Canon printer. Check the print queue for stuck jobs and cancel them. In Windows, right-click the printer icon and confirm Use Printer Online is selected. Restart the printer, wait for it to fully boot, and try printing again. If the printer still shows offline, delete it and re-add it from the Printers and Scanners menu after restarting the print spooler service.

2

Canon printer wireless setup

For most Canon PIXMA models, press and hold the Wi-Fi button on the printer until the Wi-Fi indicator flashes. Then press and hold the WPS button on your router within 2 minutes. Alternatively, use the Canon PRINT Inkjet and SELPHY app for guided wireless setup. From the printer panel on touchscreen models, go to Wi-Fi Setup, Wireless Setup Wizard, select your network, and enter the password.

3

Canon driver installation

Download the driver from usa.canon.com/support. Search for your exact Canon model number, which appears on the front panel or on a label underneath the printer. Select your operating system. Download the recommended driver and software package. Run the installer and choose your connection type when prompted. For scanning, Canon requires a separate ScanGear or ICA driver — download both if you use the scanner.

4

Canon ink absorber full error

Canon inkjet printers have an internal ink absorber pad that fills with waste ink over time. When it is full, the printer displays a message such as The ink absorber is almost full or a 5B00 error code. This requires either replacing the absorber pad or using the Canon service tool to reset the counter. This is a normal maintenance task for inkjet printers used for several years.

Canon print quality fixes

Common Canon print quality problems and how to fix them.

Streaky or missing color

Use the printer maintenance menu to run a Nozzle Check. Print the check pattern and compare it to the reference. If gaps appear in any color, run the printhead cleaning utility. Repeat the nozzle check after cleaning. Up to 3 cleaning cycles may be needed to clear clogged nozzles.

Running more than 3 cleaning cycles wastes significant ink without additional benefit. Contact support if nozzles remain clogged.

Print alignment problems

After replacing ink cartridges, print quality may be slightly off until the printer aligns the new cartridge. Go to the maintenance menu and run Print Head Alignment. Follow the on-screen instructions to select the best alignment pattern.

Run alignment after every cartridge replacement, especially if you switch between genuine and third-party cartridges.

Slow printing speed

Canon inkjet printers print slowly in High Quality mode. Switch to Standard or Fast modes for everyday documents. Also check if the printer is set to photo paper mode when printing plain paper — this dramatically reduces speed.

The Canon PRINT app also allows speed adjustment from a phone or tablet without opening a computer.

Paper feed errors

Remove all paper from the cassette. Fan the paper stack to separate sheets. Reload the paper and confirm the cassette guides are snug against the paper edges. Do not mix paper sizes or types in the same tray.

For sticky or damp paper, clean the paper feed rollers with a slightly dampened lint-free cloth.
