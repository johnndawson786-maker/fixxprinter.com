Printer setup guide

Set up your printer correctly the first time — USB, Wi-Fi, or Ethernet.

This guide covers every step of printer installation: unboxing, loading supplies, choosing a connection method, installing drivers, and confirming everything works with a test print. Works for HP, Canon, Epson, and Brother printers on Windows 10, Windows 11, and macOS.

Before you start

What you need before setting up your printer.

Gather these items before beginning. Having them ready prevents interruptions during setup.

📦

The printer itself

Remove all tape, packing foam, and protective strips inside the paper tray and around the cartridge area.

🖨️

Ink or toner cartridges

Most printers include starter cartridges. Open the ink compartment and install them before powering on.

📄

Paper

Load plain A4 or letter-size paper into the tray. Do not overfill — stay below the max fill line.

🌐

Your Wi-Fi password

For wireless setup you will need the exact SSID (network name) and password for your home or office router.

💻

Your computer or device

Have the device you want to print from nearby. You will install the driver and add the printer on it.

🔌

Power outlet access

Place the printer within reach of a power outlet and, if using USB, within cable reach of your computer.

Setup Checklist Complete installation in 4 stages

Follow each stage in order. Skipping a step is the most common reason setup fails.

Setup workflow

Step-by-step printer installation process.

1

Unbox and prepare the hardware

Remove all packaging material including tape inside the paper tray. Install ink or toner cartridges. Load paper. Power the printer on and wait for the startup process to complete. The display should show a ready or idle status.

2

Choose your connection method

USB: Connect the cable from the printer to your computer directly. Skip network setup. Wi-Fi: Use the printer's display panel to select your wireless network and enter the password. WPS: Press WPS on the router and then on the printer within 2 minutes. Ethernet: Run a network cable from the printer to your router for a stable wired connection.

3

Download and install the correct driver

Go to the official manufacturer website (hp.com/support, canon.com/support, epson.com/support, or brother.com/support). Search for your exact model number. Download the full driver package for your operating system. Run the installer and follow the prompts. Do not download drivers from third-party sites.

4

Add the printer and run a test print

On Windows: open Settings → Bluetooth & devices → Printers & scanners → Add device. On Mac: go to System Settings → Printers & Scanners → Add Printer. Select your printer from the list. Print a test page. If the test page prints clearly, setup is complete.

Common setup problems

Printer not showing up during setup? These are the most common reasons.

Printer not found on network

Confirm the printer and computer are on the same Wi-Fi network. Try restarting both the printer and router, then search again.

Driver installation fails

Run the installer as administrator. Temporarily disable antivirus during installation. Use the full driver package, not just the basic driver.

Printer connects but drops Wi-Fi

Place the printer closer to the router. Check that the printer is configured for the 2.4 GHz band rather than 5 GHz.

Setup by operating system

Platform-specific instructions for adding a printer.

Windows 10 & 11

Settings → Bluetooth & devices → Printers & scanners → Add a printer or scanner. Windows will search for available printers automatically.

If the printer is not found, click "The printer that I want isn't listed" and enter the IP address manually.

macOS Ventura / Sonoma

System Settings → Printers & Scanners → click the + button to add a printer. macOS downloads the driver automatically for most supported models.

For older printers, download the driver from the manufacturer website before adding.

iPhone & iPad (AirPrint)

No app required. Open any document or photo, tap Share → Print, and select your AirPrint-compatible printer. The device and printer must be on the same Wi-Fi network.

AirPrint is supported on most HP, Canon, Epson, and Brother models made after 2012.

Android (Mopria)

Install the manufacturer's app or use Android's built-in printing system. Go to Settings → Connected devices → Printing to add a print service.

The Google Cloud Print service has been discontinued. Use the manufacturer app for Android printing.
